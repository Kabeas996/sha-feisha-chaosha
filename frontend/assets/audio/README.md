# 游戏音效来源

本目录中的音效均来自 Kenney，许可证为 Creative Commons CC0 1.0。

- `select.ogg`、`charge.ogg`、`victory.ogg`：Kenney Interface Sounds
- `attack-kill.ogg`、`attack-flying.ogg`、`defend.ogg`：Kenney RPG Audio
- `attack-super.ogg`、`block.ogg`、`hit.ogg`：Kenney Impact Sounds

来源：

- https://kenney.nl/assets/interface-sounds
- https://kenney.nl/assets/rpg-audio
- https://kenney.nl/assets/impact-sounds
- https://creativecommons.org/publicdomain/zero/1.0/

Kenney 的 CC0 素材无需署名；这里保留来源记录，便于后续维护。
